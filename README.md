# Book Search MERN
  
## Description
  
To create a book search app using react, apollo server, and graphql.
It's still buggy.
  
## Installation
  
To install necessary dependencies, run the following command:
  
npm i
npm run build

## Usage
  
Type the "npm run start" command to run it.
